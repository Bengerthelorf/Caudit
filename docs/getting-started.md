If you don't have Claudit yet, head to [**install**](/install) first. Once
it's running you'll see a small icon in your menu bar showing today's spend.

## First Launch

Click the menu bar icon to open the popover with:

- **Usage** — today's cost, token breakdown, burn-rate estimate
- **Quota** — 5-hour and 7-day utilization with reset timers

Open the dashboard for historical data: heatmaps, session browser,
per-project / per-model / per-tool cost breakdowns.

## Requirements

- macOS 15.0 or later
- Claude Code installed (`~/.claude` directory with usage logs)
- For quota tracking: logged in via `claude auth login`
